CSS selectors

Combinators Example
A, B div, span will match both span and div elements. The , selector is a grouping method that selects all the matching nodes.
A B div span will match allspan elements that are inside a div element. 「〇〇要素の子である△△要素」と限定する書き方
A.B p.sample p要素の中でclass属性の値が「sample」となっている要素だけにスタイルが適用される。
A > B ul > li will match all lielements that are nested directly inside a ul element. 直下の子要素にだけ適応させる
A + B お隣さんにだけ適応させる
A ~ B 「+」は隣接する要素限定に作用するが、「~」は隣、または離れた要素を指定でき、指定した全てに対して適用。

A, B

AとB両方にスタイルを適用する書き方


                <p class="sample-1">スタイル適用</p>
                <p class="sample-2">スタイル適用</p>
        

                .sample-1,.sample-2 {
                    color: orange;
                  }

▼結果

スタイル適用

A B

「〇〇要素の子である△△要素」と限定する書き方

ちなみに、SCSSで書くと視覚的にわかりやすい。入れ子にするだけ。複雑になったときに便利。

例1:


                <p><span>こっちだけ適用</span></p>
 
                <ul>
                    <li>
                        <span>こっちは適用されない</span>
                    </li>
                </ul>
        

                p span {
                    color: red;
                }

▼結果

ここはデザイン反映されるよ

  • ここはデザイン反映されないよ

例2:

この子要素指定はいくらでも連続して記述することができる。


                <div><p><span>こっちだけ適用</span></p></div>
                <p><span>こっちは適用されない</span></p> 
        

                div p span {
                    color: red;
                }

▼結果

こっちだけ適用

こっちは適用されない

A.B


                <p class="sample">こっちだけ適用</p>                   
                <p>こっちは適用されない</p>     
        

            p.sample{
                color: orange;
              }

▼結果

こっちだけ適用

こっちは適用されない

A > B

直下の子要素にだけ適応させる


            <div class="example">
            <p>こっちだけ適用</p>
            <div>   
              <p>こっちは適用されない</p>
            </div>
          </div>
        

            .example > p {
                color: orange;
              }

▼結果

こっちだけ適用

こっちは適用されない

A + B

お隣さんにだけ適応させる


                <h1>これのお隣さんにだけ適応させる</h1>
                <p>ここだけ適用</p>
                <p>ここは適用されない</p>
        

            h1 + p {
                color: orange;
              }

▼結果

これのお隣さんにだけ適応させる

ここだけ適用

ここは適用されない

A ~ B

「+」は隣接する要素限定に作用するが、「~」は隣、または離れた要素を指定でき、指定した全てに対して適用。


            <h1>ここから離れたところに適用させる</h1>
                <h2>ここは適用なし</h2>
                <p>ここに適用!!</p>
                <h3>ここは適用なし</h3>
                <p>ここにも適用!!</p>
        

            h1 ~ p {
                color: orange;
              }

▼結果

ここから離れたところに適用させる

ここは適用なし

ここに適用!!

ここは適用なし

ここにも適用!!
